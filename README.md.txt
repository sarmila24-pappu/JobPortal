# 🏢 Job Portal - Spring Boot Application

A **Job Portal** application built using **Spring Boot, Hibernate, and MySQL**, allowing users to **post jobs, search for jobs, and apply**.

## 📌 Features
✅ User Registration & Login  
✅ Role-Based Access (Admin, Employer, Job Seeker)  
✅ Post & Manage Job Listings  
✅ View and Apply for Jobs  
✅ REST API Endpoints for Job Management  

## 🛠 Tech Stack
- **Backend**: Spring Boot, Spring Data JPA, Hibernate  
- **Database**: MySQL  
- **Frontend**: React / Angular (optional)  
- **Security**: Spring Security, JWT (optional)
